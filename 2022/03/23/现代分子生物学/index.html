染色体与DNA

染色体

染色体概述

  1. 染色体在遗传上起着主要作用。是遗传物质的主要载体
  2. 染色体：DNA+蛋白质，DNA只有和蛋白质包装成染色体才能保证稳定性
  3. 同一物种每条染色体携带DNA的量是一定的，不同物种或不同染色变化很大
  4. 几个名词：
    1. 染色体：在细胞分裂时观察到存在于细胞核中的棒状可染 色结构（完整），细胞分裂时，每条染色体复制生成一条与母 链完全一样的子链，形成同源染色体对。
    2. 染色质：是细胞中期核内DNA和蛋白质复合体。可以不完整
    3. 染色单体：复制时产生的染色体拷贝。此名字通常用来形容处于随后的细胞分裂期它们分开的之前的染色体。

染色体结构

真核与原核细胞染色体外观结构及在细胞内的存在方式不同

原核细胞染色体

  1. 如细菌染色体外裹着稀疏的蛋白质，这些蛋白质与DNA的折叠、DNA复制、重组及转录过程有关。容易发生突变。
  2. 原核生物没有真正的细胞核，DNA位于一个类似“核”结构的——类核体上
  3. 细菌DNA是一条相对分子质量在$10^9$左右的共价闭合双链分子，成环状，通常也称为染色体，几乎没有蛋白质。
  4. 大多数原核生物染色体是单倍的，快速生长时可能有多条

原核生物DNA特征

  1. 原核生物中一般只有一条染色体且大都带有单拷贝基因，只有少数基因(如rRNA基因)以多拷贝形式存在
  2. 整个染色体DNA几乎全部由功能基因与调控序列所组成，几乎全部编码，少有冗余不参与基因表达的片段
  3. 几乎每个基因序列都与它所编码的蛋白质序列呈线性对应状态

真核细胞染色体

  1. 真核细胞的染色体中，DNA与蛋白质完全融合在 一起，包括：DNA、组蛋白和非组蛋白及部分RNA（主要是尚未转录完成的mRNA）
  2. 真核细胞有明显的核结构。
  3. 真核生物体细胞的染色体都是二倍体，而性细胞即生殖细胞的染色体数目是体细胞的一半，称为单倍体。
  4. 二倍体阶段的每个基因都有两个拷贝，其中的一个拷贝会通过配子，即性细胞(精子或卵子)从亲本传到子代。

真核生物染色体特征

  1. 分子结构稳定
  2. 能自我复制，使亲代与子代之间保持连续性
  3. 能指导蛋白合成，控制生命过程
  4. 能够产生遗传变异

真核细胞染色体组成

蛋白质（组蛋白、非组蛋白）+真核生物基因组DNA

组蛋白

  1. 组蛋白是染色体的结构蛋白，它与DNA组成核小体 。 通 常 可 以 用 2mol/L NaCl或 0.25mol/L的 HCl/H2SO4处理使组蛋白与DNA分开。
  2. 根据凝胶电泳性质组蛋白分为：
    1. H1：分离易，不保守，位于接头位置，数量只有其余4个组蛋白的一半
    2. H2A、H2B：分离较难，较保守，位于核心位置
    3. H3、H4：分离最难，最保守，位于核心位置

组蛋白的一般特性

  1. 进化上的极端保守性。不同物种组蛋白十分相似
  2. 无组织特异性。大多数情况下，不同组织细胞组蛋白相同
  3. 肽链上氨基酸发布的不对称性。
  4. 组蛋白的修饰作用。 由于组蛋白富含Arg、Lys，可以发生甲基化等多种翻译后修饰，可以影响翻译遗传
    1. H3、H4的修饰作用比较普遍，以甲基化、乙酰化修饰为主
    2. H2A、H2B能 发生泛素化和乙酰化修饰，H1有泛素化和磷酸化修饰。
    3. 修饰包括：包括甲基化、乙基化、磷酸化及ADP核糖基化等
      1. 甲基化：由组蛋白甲基转移酶和去甲基化酶催化的可逆修饰。
      2. 乙酰化：主要发生在核心组蛋白上。
      3. 泛素化：修饰位点为高度保守的赖氨酸残基，组蛋白H1、H2A、H2B 和H3都能发生泛素 化修饰。（一般蛋白泛素化是为了降解）
    4. 修饰只发生在细胞周期的特定时间和组蛋白的特定位点
    5. 作用：使DNA与组织蛋白之间的作用强度发生变化，进而使DNA与转录因子接触的难易度改变，影响转录作用的速率。

非组蛋白

  1. HMG蛋白：高速泳动蛋白，有HMG1和HMG2 。能与DNA结合，也能与H1作用，但与DNA的结合并不牢固，可能与DNA的超螺旋结构 有关，在DNA复制中和重组中起重要作用。
  2. DNA结合蛋白：是一种能优先与DNA结合，并使其扭曲的高移动性群蛋白。这类蛋白质可以改变核小体的排列方 式，产生更复杂的染色质结构。可能是一些与DNA的复制或转录有关的酶或调节 物质。
  3. 非组蛋白功能：
    1. 酶（RNA合成酶、蛋白磷酸化酶等）
    2. 遗传信息的保持和表达调节
    3. 染色体的结构支持体

真核生物基因组DNA

  1. 真核细胞基因组中含有大量的重复序列，而且功能DNA序列大多被编码蛋白质的非功能DNA 所隔开。
  2. 基因密度：每MB（$10^6$）基因组DNA上含有平均基因数目，与生物体高级性成负相关
  3. 基因组大小：一种生物单倍染色体中 DNA 的总长度，一般与生物体复杂性成正相关

真核细胞的DNA 序列的分类

  1. 不重复序列：一般只有一个或几个拷贝，结构基因基本属于不重复序列，如蛋清蛋白、 蚕丝心蛋白、血红蛋白和珠蛋白等都是单拷贝基因。
  2. 中度重复序列：重复次数在$10^1 \text{~} 10^4$，如，各种rRNA、 tRNA 以及某些结构基因如组蛋白基因
  3. 高度重复序列——卫星DNA：用CsCl密度梯度离心将卫星DNA与其他DNA分开， 形成两个以上的峰，即含量较大的主峰和高度重复序列小峰，后者又称卫星区带(峰)。

真核生物基因组的结构特点

  1. 真核基因组庞大，一般都远大于原核生物的基因组。
  2. 真核基因组存在大量的重复序列。
  3. 真核基因组的大部分为非编码序列，占整个基 因组序列的90%以上，该特点是真核生物与细菌和病毒之间最主要的区别。
  4. 真核基因组的转录产物为单顺反子（mRNA中1个起点和终点编码1个基因）。
  5. 真核基因是断裂基因，有内含子结构。
  6. 真核基因组存在大量的顺式作用元件。包括启动子、增强子、沉默子等。
  7. 真核基因组中存在大量的DNA多态性。
    • DNA 多态性是指DNA序列中发生变异而导致的个体间核苷酸序列的差异，主要包括单核苷酸多态性和串连重复序列多态性两类
  8. 真核基因组具有端粒结构。（原核生物为环状）
    • 由于真核生物基因的线形结构，末端无法提供翻译模版，需要有特殊结构
    • 端粒是真核生物线性基因组DNA末端的一种特殊结构，它是一段DNA序列和蛋白质形 成的复合体。其DNA序列相当保守，一般有多个短寡核苷酸串连在一起构成。
    • 端粒具有保护线性DNA的完整复制、保护染色体末端和决定细胞的寿命等功能。

附原核生物基因组DNA结构特点

  1. 结构简练
    • 几乎所有DNA参与编码蛋白质，少数不转录参与控制基因表达
  2. 存在转录单元
    1. 原核生物DNA序列中功能相关的RNA和蛋白质基因，往往丛集在基因组的一个或几个特定部位， 形成功能单位或录单元，它们可被一起转录为含多个mRNA的分子，叫多顺反子mRNA（多个基因共用一个起终点）。
  3. 有重叠基因
    • 在一些细菌和动物病毒中有重叠基因，即同一段DNA能携带两种不同蛋白质的信息。

染色质和核小体

  1. 染色质（DNA+组蛋白）是许多核小体连成的念珠状结构
  2. 染色体由DNA和蛋白质组成的证据（染色质的稳定性要大于自由DNA）：
    1. 染色质DNA的Tm值比自由DNA高
    2. 在染色质状态下，由DNA聚合酶和RNA聚合酶催 化的DNA复制和转录活性大大低于在自由DNA中的反应
    3. DNA酶I (DNaseI)对染色质DNA的消化远远慢于对纯DNA的作用。

核小体

  1. 核小体是由H2A、H2B、H3、H4各两个分子生成的八聚体和由大约200 bp DNA组成的。（不包括H1）
  2. 八聚体在中间，DNA 分子盘绕在外
  3. 每个核心组蛋白有一个N端延伸的“尾巴”，上面有许多高度修饰的位点
  4. 核小体的形成是染色体中DNA压缩的第一个阶段。组蛋白H1 通过与结合在核小体上的DNA结合，使染色质进一步压缩，形成30nm 螺线管。
  5. 染色殖是一个多层次螺旋结构，上述螺线管可进一步压缩为超螺旋结构
  6. 超螺旋圆筒进一步压缩为染色单体，整个压缩过程接近一万步

DNA

DNA概述

  • DNA或RNA都是以核苷酸为基本单位
  • 核苷酸：磷酸+核糖（戊糖）+碱基

DNA一级结构

  1. DNA的一级结构，就是指4种核苷酸的连接及其排列顺序，表示了该DNA分子的化学构成。
  2. 基本单位是脱氧核苷酸。
  3. DNA是许多核苷酸依靠3′-5′磷酸二酯链连接，每个核苷酸又由磷酸、碱基、戊糖构成、碱基又是由A、T、C、G四种成分构成。
  4. DNA通常以线性或环状形式存在，大多数是由两条碱基互补的单链构成，有少部分生物以单链形式存在。

DNA二级结构

  1. DNA 的二级结构是指两条多核苷酸链反向平行 盘绕所生成的双螺旋结构。
  2. 基本特点：
    1. DNA分子是由两条互相平行的脱氧核苷酸长链盘绕而成的。
    2. DNA分子中的脱氧核糖和磷酸交替连接，排 在外侧，构成基本骨架，碱基排列在内侧。
  3. DNA二级结构的稳定作用力：氢键、疏水作用力、范德华力、盐键

DNA高级结构

  1. DNA的高级结构:DNA的高级结构是指DNA双 螺旋进一步扭曲盘绕所形成的特定空间结构。
  2. 超螺旋结构是DNA高级结构的主要形式，可分为正超螺旋与负超螺旋两大类。
  3. 负超螺旋是细胞内常见的DNA高级结构形式，正超螺旋是过度缠绕的双螺旋。
  4. 正超螺旋与负超螺旋在不同类型的拓扑异构酶作 用下或在特殊情况下可以相互转变。

DNA双链变性与复性

  1. DNA变性：当DNA溶液温度接近沸点或者pH较高时，互补的两条链就可能分开
  2. 变性是可逆的
  3. DNA熔点Tm：把吸光度增加到最大值一半时的温度
  4. Tm的影响因素：
    1. G+C碱基的含量，G+C的含量越高，DNA的Tm值也越高。
    2. Tm =69.3+0.41(G+C)%，可以通过Tm推算碱基占比
    3. 溶液离子强度，高离子强度下，稳定，低离子浓度下，稳定性降低
    4. DNA的均一性，均质DNA的Tm范围小，异质DNA的Tm范围大

DNA的复制

  1. 复制包括：起始、延伸、终止三个阶段
  2. 原核生物和真核生物的复制过程都需要有多种酶 和蛋白质的协同参与。
  3. 采取半保留复制
    1. DNA 在复制过程中碱基间的氢键断裂，双螺旋解旋并被分开，每条链分别作为模板合成新链，产 生互补的两条链。
    2. 每个子代分子的一条链来自亲代DNA，另一条链 则是新合成的，这种复制方式被称为DNA 的半保留复制

DNA复制的一些基本概念

  1. 复制子：生物体内能独立进行复制的单位
    • 从复制起始点到终止点的区域为一个复制子
    • 一个复制子只含一个复制起始点
  2. 复制叉：复制时，双链DNA 要解开成两股链分别进行，所以，这个复制起始点呈现叉子的形式
    • 复制叉从复制起点开始沿着DNA 链连续移动
    • 起始点可以启动单向复制或者双向复制，这主要取决于在复制起点形成一个复制叉还是两个复制叉。
  3. 复制起始点
    1. 细菌、酵母、线粒体和叶绿体中的复制起始点的共同特点是含有丰富的AT序列，它可能有利于DNA复制启动时双链的解开。
    2. 细菌、病毒和线粒体的DNA 分子都是作为单个复制子完成复制的
    3. 真核生物基因组可以同时在 多个复制起点上进行双向复制。
  4. 复制终止点
    1. 复制子中控制复制终止的位点称为复制终止点
    2. 对于原核生物的环形DNA，复制终止点为两个复制叉的相遇位点
  5. 复制方向
    1. 单向复制：从一个复制起始点开始，只有一个复制叉在移动。某些环状DNA利用这种方式拷贝双 链DNA，如质粒ColE1 DNA。
    2. 双向复制：复制起始于一个位点，但向两侧分别形成复制叉， 向相反方向移动。这种复制方式最为普遍。无论是原核生物还是真核生物，DNA的复制主要是从固定的起始点以双向等速复制。
    3. 相向复制：从两个起始点分别起始两条链的复制，这种模式 虽然有两个复制叉的生长端，但在每个复制叉中只有一条链作为模板合成DNA。如某些线形DNA病毒
  6. 复制速度
    1. 真核生物基因组庞大，有高级结构，复制时需要解开核小体，复制速度较慢
    2. 但真核生物有多个复制起始点，可以多线程同步进行复制子的复制
  7. 冈崎片段与半不连续复制：
    1. 由于DNA聚合酶只能从母链的 $3’ \rightarrow 5’$ 端开始复制，合成链方向：$5’ \rightarrow 3’$ ，新合成的两条链分位：前导链和后随链
    2. 前导链合成连续，后随链不连续
    3. 复制叉由5’向3’方向（新链）连续复制，称为前导链；另一条链复制叉由3’向5’移动，而DNA复制方向不变，形成许多不连续片段，称为冈崎片段，最后连接成完整的DNA，称为后随链。
  8. DNA聚合酶的作用机制：
    • 3个结构域：拇指、手指和手掌

原核生物DNA复制的特点

大肠杆菌基因组以双链环状DNA分子的形式存在， 复制起始区位于其遗传图的84 min附近，复制起 始点(oriC)含有3个13 bp的串联重复保守序列，以 及4个由9 bp的保守序列组成的能结合DnaA的起始结合位点。

  1. DNA双螺旋的解旋
    • DNA在复制时，首先在拓扑异构酶I的作用下解开负超螺旋，并与解链酶共同作用，在复制起始点处解开双链。SSB蛋白来稳定解开的单链，接着由引发酶等组成的引发体迅速作用于两条单链 DNA上。
    • 前导链和后随链的合成，都需要一段RNA引物以起始子链DNA 的合成。
    • DNA解链酶、单链结合蛋白（SSB）、DNA拓扑异构酶（消除正超螺旋累计）
  2. DNA复制的引发
    • DNA复制时，往往先由引发酶在DNA模板上合成一段RNA链，作为引发末端(引物)。
    • 前导链引发过程比较简单，只要有一段RNA引物;
    • 后随链的引发过程由引发体引发。
  3. 复制的延伸
    • 在复制的延伸过程中，前导链和后随链的合成同时进行。
    • 前导链持续合成，合成方向与复制叉一致。
    • 后随链的合成分段进行，形成中间产物冈崎片段， 再通过共价连接成一条连续完整的新DNA 链。
  4. 复制的终止
    • 当复制叉前移，遇到约22个碱基的重复性终止子序列(Ter)时，Ter-Tus复合物能使DnaB不再将 DNA解链，阻挡复制叉的继续前移，等到相反方 向的复制叉到达后停止复制，其间仍有50~100 bp 未被复制，由DNA修复机制填补空缺，其后两条链解开。
  5. DNA聚合酶（原核生物中，以大肠杆菌为例）
    1. DNA聚合酶I：不是复制染色体的主要聚合酶，该蛋白可以被蛋白酶切成两个区域：
      • C端区域（Klenow片段）同时具有DNA聚合酶活性和3’—>5‘外切酶活性，可以合成DNA链又能降解DNA，保证DNA复制的准确性。
      • N端区域具有5‘—>3’外切酶活性，可作用于双链DNA，可水解5‘端或5’端几个核苷酸处的磷酸二酯键，因而可以在切除由紫外线照射形成的嘧啶二聚体中起重要作用。也可以除去冈崎片段5‘端RNA引物，使冈崎片段缺口消失，保证连接酶能连接片段
    2. DNA聚合酶II：有5‘—>3’聚合酶活性，但活性低，不是主要复制酶。有3‘—>5’外切核酸酶活性起校正作用。主要是修复DNA作用
    3. DNA聚合酶III：有5‘—>3’聚合酶活性，有3‘—>5’外切核酸酶活性，活性很强，能在引物的3‘-OH上5万个核苷酸/min的速度延长DNA链，是延长链的主导酶
    4. 共同点：均以 dNTP 为底物,需Mg- 激活，聚合时必须有模板链和具有3’-0H 末端的引物链，链的延伸方向为5’—>3’。
  6. 原核生物DNA复制的调控
    1. 迅速分裂的细胞具较多复制叉，而分裂缓慢的细胞复制叉较少并出现复制的间隙。
    2. 细胞内复制叉的多少决定复制起始频率的高低， 这可能是原核细胞复制的调控机制。
    3. 复制起始频率的直接调控因子是蛋白质和RNA。

真核生物DNA复制特点

  1. 真核生物 DNA每个细胞周期只精确地复制一次。
    • 在真核生物细胞中，DNA复制只是细胞周期的一 部分，它只在S期进行。第一批复制子的激活标 志着S 期的开始。
  2. 真核细胞复制的起始需要在前复制复合体pre-RC的指导下进行。
    • 酵母的复制起始点称为自主复制序列
    • 对指导复制起始的序列进行识别(复制器选择)发生在G1期。
    • 由4个独立的蛋白质组成的前复制复合体pre-RC 的 形成介导了复制器的选择。
  3. 细胞周期蛋白依赖性激酶调控前复制复合体的形成和激活
    • 只有在细胞从G1到达S期后，由于调控pre-RC的 两种激酶Cdk和Ddk在S期被激活，pre-RC被磷酸化后诱发起始点上其他复制蛋白的组装和复制的 起始。
  4. 真核生物DNA聚合酶
    1. 真核细胞的 DNA 聚合酶和细菌 DNA 聚合酶基本性质相同,均以 dNTP 为底物,需Mg- 激活，聚合时必须有模板链和具有3’-0H 末端的引物链，链的延伸方向为5’一3’。
    2. 但真核细胞的 DNA 聚合酶一般都不具有外切核酸酶活性，推测一定有另外的酶在 DNA复制中起校对作用。
    3. 真核生物DNA复制比较公认的说法:在复制叉上存在一个DNA聚合酶α/引发酶复合体和另外两个 DNA聚合酶复合体。这两个聚合酶中一个是DNA 聚合酶δ，另一个是第二个DNA聚合酶δ或DNA聚合酶ε，前者延伸前导链，后者合成后随链的冈崎片段
    4. 冈崎片段RNA 引物的去除分为两步:
      • 首先，一种可特异性切除DNA-RNA杂合底物的 RNA酶H1发挥内切核酸酶的活性，在靠近RNA 与DNA的连接处切开引物，由具备5′→3′外切核 酸酶活性的FEN1蛋白降解RNA片段。
      • 最后，由DNA连接酶I将相邻的冈崎片段连接起来。
  5. 端粒酶与DNA末端复制
    1. 所有新的DNA合成启动都需要一条引物，这使线性染色体末端的复制成为难题
    2. 用蛋白质代替RNA作为每个染色体末端最后一个冈崎片段的引物。
    3. 多数真核细胞利用端粒酶来延伸染色体3′端，解决末端复制问题。端粒酶由蛋白质和RNA组成。 端粒酶利用其自身含有的RNA成分作为模板将端 粒序列添加到染色体3′ 端。

  6. 真核细胞DNA复制调控

    1. 细胞生活周期水平调控 也称为限制点调控，即决定细胞停留在G1 期还是进入S 期。
    2. 染色体水平调控 决定不同染色体或同一染色体不
    3. 同部位的复制子按一定顺序在S 期起始复制。 3 复制子水平调控 决定的起始与否。

      真核细胞DNA的复制调控

  8. 染色体水平调控： 决定不同染色体或同一染色体不同部位的复制子按一定顺序在S 期起始复制。

DNA的突变与修复

DNA突变

基因突变：是指在基因内的遗传物质发生可遗传的 结构和数量的变化。
DNA的突变：主要来源于DNA复制过程中出现的错误、遗传物质的化学和物理损伤，以及转座子插入所造成的DNA 序列变化。
复制错误合DNA损伤的两个后果：

  • 一是给 DNA 带来永久性的不可逆的改变，最终 改变基因编码的序列，称之为基因突变;
  • 二是 DNA 的某些化学变化使得 DNA 不能再被用 作模板进行复制和转录。

    DNA损伤的修复

DNA的转座

  • DNA的转座，或称移位是由可移位因子介导的遗传物质重排现象。
  • 在转座的过程中，可移位因子的一个拷贝常常留 在原来的位置上，在新位点上出现的仅仅是拷贝。转座有别与同源重组，它依赖于DNA的复制。
  1. 转座子（Tn）的分类和结构特征
    1. 转座子是存在于染色体DNA上可自主复制和位移的基本单位。
    2. 分类：插入序列、复合型转座子

      SNP理论与应用

      单核苷酸多态性(SNP)，指基因组DNA 序列中由于单个核苷酸 (A、T、C 和G)的突变而引起的多态性。
      SNP是基因组中最简单、最常见的多态性形式， 具有很高的遗传稳定性。

生物信息的传递——从DNA到RNA

RNA的功能

  1. DNA和蛋白质之间的桥梁作用
  2. 生物功能分子

    DNA与RNA的差异

  3. 组成单位不同:DNA的组成单位是脱氧核苷酸，RNA的组成单位是核糖核苷酸
  4. 组成碱基不同:DNA的组成碱基是ATGC，RNA的组 成碱基是AUGC
  5. 组成五碳糖不同:DNA的组成五碳糖是脱氧核糖， RNA的组成五碳糖是核糖
  6. 空间结构不同:DNA是双螺旋结构，RNA一般是单链

    基因的表达

  7. DNA序列是遗传信息的贮存者，它通过自主复制 得到永存，并通过转录生成信使RNA，翻译生成蛋白质的过程来控制生命现象。
  8. 基因表达包括转录(transcription)和翻译(translation)两个阶段。
    1. 转录是指拷贝出一条与DNA链序列完全相同(除了 T→U之外)的RNA单链的过程，是基因表达的核心步骤。
    2. 翻译是指以新生的mRNA为模板，把核苷酸三联遗传密码子翻译成氨基酸序列、合成多肽链的过程，是基因表达的最终目的。
  9. 贮存于DNA双链中的遗传信息通过转录按照碱基互补配对的原则被转化成为单链RNA分子。

RNA合成的特点

  1. RNA 是以5’→3’方向合成的，它的序列是与DNA编码链(意义链)相同。
  2. RNA 的合成是以反义链(模板链)为模板。
  3. 同在DNA中一样，形成磷酸二脂键(Phosphodiester bonds)。
  4. DNA双链中，与mRNA序列相同 的那条DNA链称为编码链(coding strand)或称有义链(sense strand)，另一条根据碱基互补原则指导 mRNA合成的DNA链称为模板链template strand)或称反义链

RNA结构、分类和功能

  1. mRNA生物合成过程和成熟mRNA结构在原核细胞和真核细胞是不同的。
  2. 真核细胞往往以一个较大相对分子质量的前体RNA出现在核内，需要经过转录后加工。
  3. 原核生物以AUG(有时GUG，甚至UUG)作为起始密码子，真核生物以AUG 作为起始密码子。

RNA结构

RNA是单链，但会自身折叠，形成局部双螺旋
RNA以 多 种 茎 -环 结 构 (stem-loopstructure)， 如 发 夹 结 构 (hairpin)、 凸 起 (bulge)、 环 状 结 构 (loop)或 四通道内环(four-stemjunction)的形式存在。还可以形成假结

RNA分类

生物体内主要有3 种RNA参与遗传信息的传递：

  1. 编码特定蛋白质序列的信使RNA(messenger RNA,mRNA)
  2. 能特异性解读mRNA中的遗传信息、将其转化成 相应氨基酸后加入多肽链中的转运RNA(transfer RNA, tRNA)
  3. 直接参与核糖体中蛋白质合成的核糖体 RNA(ribosomal RNA, rRNA)。
    非编码RNA

    RNA的功能

  4. 作为细胞内蛋白质生物合成的主要参与者;
  5. 部分RNA可以作为核酶在细胞中催化一些重要的反应，主要作用于初始转录产物的剪接加工;
  6. 参与基因表达的调控，与生长发育密切相关;
  7. 在某些病毒中，RNA是遗传物质。

RNA转录

RNA转录和DNA复制的比较

RNA转录与DNA的复制在化学和酶学上非常相似，但存在一些明显的差别:

  1. RNA聚合酶可从头起始转录，在催化RNA合成
    时不需要引物。
  2. 添加几个核苷酸后，正在延长的链脱离模板。因此，多个RNA聚合酶可以同时转录一个基因， 在短时间内合成大量转录产物。
  3. 转录具有选择性。RNA转录则是选择性地复制基因组的特定部分，可以产生几个到上千个相同的拷贝。
  4. 转录过程缺乏严谨的矫正机制。转录中每添加10000个 核苷酸会发生一次错误，与DNA复制的精确度相比差错率较高。

转录机器的主要成分——RNA聚合酶

  1. RNA聚合酶是转录过程中最关键的酶。
  2. 原核和真核生物的RNA聚合酶虽然都能催化RNA 的合成，但在其分子组成、种类和生化特性上各

启动子与转录起始

  1. 启动子是一段位于结构基因5′端上游区的DNA序列，能活化RNA聚合酶，使之与模板DNA准确地 相结合并具有转录起始的特异性。
  2. 转录的起始是基因表达的关键阶段，而这一阶段 的重要问题是RNA聚合酶与启动子的相互作用。启动子的结构影响了它与RNA聚合酶的亲和力
  3. 转录单位(transcription unit)是一段从启动子开始 至终止子(terminator)结束的DNA序列，RNA聚合酶从转录起点开始沿着模板前进，直到终止子为止，转录出一条RNA链。
  4. 转录起点是指与新生RNA链第一个核苷酸相对 应DNA链上的碱基，通常为一个嘌呤。
    1. 常把起点前面，即5′端的序列称为上游序列，而把其后面即3′端的序列称为下游序列。
    2. 在描述碱基的位置时，一般用数字表示，起点为+1，下游方向依次为+2、+3……等，上游方向依次为-1、-2、-3……等。

转录的基本过程

RNA链合成特点：

  1. RNA按 5′→3′ 方向合成，以DNA双链中的反义链(模板链)为模板，在RNA聚合酶催化下，以4种 核苷三磷酸(NTPs)为原料，根据碱基配对原则，各核苷酸间通过形成磷酸二酯键相连，不需要引物的参与，合成的RNA带有与DNA编码链相同的序列(A-U)。

    基本过程：

  2. 模版识别
    1. 模板识别(template recognition)阶段主要指RNA聚合酶识别启动子序列并与启动子DNA双链特异性结合的过程。
    2. 启动子(promoter)是基因转录起始所必需的一段DNA序列，是基因表达调控的上游顺式作用元件之一。
  3. 转录起始
    1. 转录起始(initiation)不需要引物。RNA聚合酶结合在启动子上以后，使启动子附近的DNA双链 解旋并解链，形成转录泡以促使底物核糖核苷酸与模板DNA的碱基配对。
    2. 转录起始就是RNA链上第一个核苷酸键的产生。
  4. 转录延伸
    1. 进入转录延伸(elongation)阶段，在解链区NTP不 断被添加到新生RNA链的3′-OH端，形成RNA- DNA杂合物。随着转录复合体沿着DNA模板向前 移动，DNA双螺旋持续解开，暴露出新的单链 DNA模板，RNA链不断延伸。在解链区的后面， DNA模板链与其原先配对的非模板链重新结合成 为双螺旋，RNA链被逐步释放。
    2. DNA转录循环假说解释RNA链的延伸：首先，三磷酸核苷酸(NTP)填补了开放的底物位点 并在活性位点形成磷酯键;此时，处于 RNA聚合酶II活性位点的核酸发生“移位”，其中连接区的α -螺旋结构从笔直变为弯折再恢复为笔直状态，为 下一轮RNA合成留出了空的底物位点。
  5. 转录终止
    1. 当RNA链延伸到转录终止(termination)位点时， RNA聚合酶不再形成新的磷酸二酯键，RNA- DNA杂合物分离，转录泡瓦解，DNA恢复成 双链状态，RNA聚合酶和RNA链都被从模板上释放出来，转录终止。

原核与真核生物mRNA转录特征比较

  1. 转录和翻译
  • 原核生物中，mRNA的转录和翻译不仅发生在同一个细胞空间里，而且这两个过程几乎同步 。转录和翻译是偶联在一起的
  • 真核细胞的mRNA往往以较大分子量前体RNA出现在核内，只有成熟的、相对分子质量明显变 小并经化学修饰的mRNA才能进入细胞质，参与蛋白质的合成。所以，真核细胞mRNA的合成和 功能表达发生在不同的空间和时间范畴内。
  1. 过程比较
    1. 原核生物：只有1种RNA聚合酶参与转录。
    2. 真核生物：3种以上RNA聚合酶参与不同类型的基因转录，合成不同类型不同定位的RNA
  2. 转录产物：
    1. 原核生物的初始转录产物大多 数是编码序列，与蛋白质的氨基酸序列呈线性关系
    2. 真核生物的初始转录产物含有内含子序列，成熟的 mRNA只占初始转录产物的一小部分。
  3. 初始转录：
    1. 原核生物初始转录几乎不需要剪接加工，就可直接作为成熟的mRNA进一步行使翻译模板的功能
    2. 真核生物转录产物需要经过剪接、修饰等转录后加工成熟过程才能成为成熟的mRNA。
  4. 偶联性：原核生物转录翻译在同一空间，几乎同步进行。真核生物转录翻译在不同时间和空间

原核生物mRNA的特征

  1. 原核生物mRNA的半衰期短
    1. 细菌基因的转录，翻译是紧密相联的，蛋白合成时，mRNA的3‘端还远远没有转录完全
    2. mRNA的降解紧随着翻译
  2. 许多原核生物mRNA以多顺反子
    1. 只编码一个蛋白质的mRNA称为单顺反子;编码多个蛋白质 的 mRNA称为多顺反子 mRNA
    2. 细菌mRNA可以同时编码不同的蛋白质，多顺反子mRNA是一组相邻或相互重叠基因的转录产物，这样的一组基因可被称为一个操纵子
    3. 顺反子(cistron)即结构基因，为决定一条多肽链合成的功能单位。顺反子的概念来自遗传学中的顺反重组试验，是确定交换片段究竟在一个基因内还是属于两个基因的试验，顺反子在一定条件下与基因同义，单顺反子是一个基因，多顺反子是多个基因。
    4. 两个顺反子距离较远，翻译的终止和起始是独立的。距离较近时，前一顺反子的终止与后一顺反子起始衔接
    5. 第一个蛋白合成终止后，核糖体分解成大、小亚基，脱离mRNA模版
    6. 前一个多肽翻译完成后，核糖体大、小亚基分离，小亚基不离开mRNA模版，启动下一个多肽
  3. 原核生物mRNA的5’端无帽子结构，3‘端没有或只有较短的多（A）结构
    1. 原核生物起始密码子AUG上游7~12个核苷酸处 有被称为Ribosome Binding Site (RBS)或SD序列的保守区，该序列与 16S rRNA 3′端反向互补，被认为在核糖体与mRNA的结合过程中起作用。

真核生物mRNA的特征

  1. 真核基因几乎都是单顺反子，只包含一个蛋白质的信息，其长度在几百到几千个核苷酸之间。
  2. 一个完整基因，包括编码区，5′ 和 3′ 端长度不等的特异性序列。
  3. 真核生物mRNA结构上的最大特征是5′ 端的帽子 及3′ 的多(A)结构。
真核生物mRNA的5′ 端存在“帽子”结构
  1. 真核生物(不包括叶绿体和线粒体)mRNA的5′端都是经过修饰的，基因转录一般从嘌呤(主要是A， 或G)起始，第一个核苷酸保留了5′端的三磷酸基 团，其3′-OH位与下一个核苷酸的5′磷酸基团形成 二酯键，转录产物的起始序列为pppApNpNp……。
  2. 5′终端是一个在mRNA转录后加上去的甲基化鸟嘌呤核苷酸。
  3. 帽子结构可能使mRNA免遭核酸酶破坏
  4. mRNA的帽子结构通常被甲基化
  5. 帽子功能：
    1. 有助于mRNA越过核膜，进入胞质;
    2. 保护5′不被核酶降解;
    3. 翻译时供IFIII(起始因子)和核糖体识 别，是翻译所必需的。
绝大多数真核生物mRNA具有多聚(A)尾巴
  1. 除组蛋白基因外，真核生物mRNA的3’末端都有多 聚(A)序列，其长度因mRNA种类不同而变化， 一般为40-200个左右。它是在转录后加上的
  2. 几乎所有真核基因的3′端转录终止位点上游15~30 bp处 的保守序列AAUAAA对于初级转录产物的准确切割及 加多(A)是必需的。
  3. 多(A)位点及AATAAA的存在对于基因的转录和成熟意义重大。
  4. 多聚（A）的功能：
    1. 是mRNA由细胞核计入细胞核的必需形式
    2. 大大提高了mRNA在胞质中的稳定性
    3. 可促进核糖体的有效循环
  5. 尽管大部分真核mRNA有poly (A)尾 巴，细胞中仍有多大1/3没有poly (A) 的mRNA，将其称为Poly (A) –，约1/3的Poly (A) –mRNA编码了不同 形式的组蛋白（组蛋白在核中且保守）。

原核生物RNA聚合酶及其转录

原核生物RNA聚合酶

  1. 在细菌中，一种RNA聚合酶负责所有mRNA、 rRNA和tRNA的合成。
  2. 大多数原核生物RNA聚合酶的组成相同，大肠杆菌RNA聚合酶由2个α亚基、一个β亚基、一个β′ 亚基和一个ω亚基组成核心酶，然后加上一个σ 亚基后则成为聚合酶全酶
  3. RNA聚合酶是转录过程中最关键的酶。
    1. 主要以双链DNA为模板，以4种核苷三磷酸作为活性前体。
    2. 需要Mg2+/Mn2+为辅助因子。
    3. 它不需要任何引物。
    4. 以5’ —> 3’ 方向合成RNA链。
    5. 缺乏3’ —> 5’ 外切酶活性。
    6. 是一个含有多个亚单位的酶。
  4. 原核生物转录的起始需要全酶参与，由σ因子辨认起始点，延长过程仅需要核心酶的催化。
  5. 转录的准确性取决于特异的转录起始位点，转录起始后按照碱基互补原则准确地转录模板DNA序列及具有特异的终止部位。

原核生物启动子

原核生物启动子结构特点：在被保护区内有一个由5个核苷酸组成的共同序 列，是RNA聚合酶的紧密结合点，现在称为 Pribnow区(Pribnow box)，这个区的中央大约位 于起点上游10 bp 处，所以又称为−10区。
绝大多数启动子都存在两端共同序列：

  1. -10 bp处的TATA区
  2. -35 bp处的TTGACA区
    是RNA Pol与启动子的结合位点，与σ因子有很高的亲和力
    RNA Pol对启动子的识别，并不是识别碱基本身，而是通过氢键互补的方式识别，保守区某些碱基的取代可能不会影响启动子的功能。

原核生物RNA转录周期

原核生物RNA的转录周期可以分为转录起始、延伸和终止过程。

RNA转录起始
  1. RNA聚合酶全酶对启动子的识别，聚合酶与启动 子可逆性结合形成封闭复合物。此 时，DNA链仍处于双链状态。
  2. 伴随着DNA构象上的重大变化，封闭复合物转变成开放复合物，聚合酶全酶所结合的 DNA序列中有一小段双链被解开。
  3. 开放复合物与最初的两个NTP相结合，并在这两 个核苷酸之间形成磷酸二酯键后转变成包括RNA聚合酶 、 DNA和 新 生 RNA的 三元复合物
新生RNA链的延伸
  1. 新生的RNA链的长度达到9~10个核苷酸时，σ因子从RNA聚合酶全酶上脱落下来，核心酶沿 模板DNA链移动并使新生RNA链不断伸长的过程就是转录的延伸
  2. 延伸阶段，DNA和聚合酶分子都发生了构象变化。由核心酶、DNA和新生RNA所组成的转录延伸复合物是转录循环中一个十分重要的环节。
  3. 延伸RNA聚合酶同时具有合成和校对两种功能，校对机制：焦磷酸编辑、水解编辑
RNA转录的终止
  1. RNA聚合酶起始基因转录后，它就会沿着模板5′ → 3′ 方向不停地移动，合成RNA链，直到碰上终止信号时，RNA聚合酶才停止加入新的核苷酸， 与模板DNA相脱离并释放新生RNA链。
  2. 终止时，RNA-DNA杂合体的氢键必须被破坏，DNA双链才能重新组合
  3. 根据RNA聚合酶是否需要辅助因子参与才能终止 RNA链的延伸，可将大肠杆菌中的终止子分为不 依赖于ρ因子和依赖于ρ因子两大类：
    1. 在不依赖于ρ因子这类终止反应中，没有任何其 他因子参与，核心酶就能终止基因转录。模板DNA上存在终止转录的特殊信号——终止子，又称内在或固有终止子
    2. 内在终止子的两个明显结构：终止位点上游一般存在一个富含GC碱基的反向重复序列，由这段DNA转录 产生的RNA容易形成发夹结构。 在终止位点前面有一段由4~8个A-T碱基对序列，其转录产物的3′ 端为寡聚U。
    3. 依赖于ρ因子的终止子有些终止位点的DNA序列缺乏共性，而且不能形成强的发夹结构，因而不能诱导转录的自发终止。RNA聚合酶不能识别这 些转录终止信号而停止转录，只有在加入ρ因子后该聚合酶才能在DNA模板上准确地停止转录。
    4. ρ因子由6个相同亚基组成，具有NTP酶和解螺旋酶活性，通过催化NTP的水解促使新生RNA链从三元转录复合物中解离出来，终止转录

真核生物RNA Pol与RNA转录

真核生物RNA Pol

真核生物中共有3类结构相对更复杂的RNA聚合酶，它们在细胞核中的位置不同，负责转录的基因不同，对 α- 鹅膏蕈碱的敏感性也不同 。
线粒体和叶绿体有各自独立的RNA Pol

  1. RNA聚合酶I：位于核仁中，转录产物是除5S rRNA的各种rRNA，对鹅膏蕈碱不敏感
  2. RNA聚合酶II：位于细胞核质内，在核内转录生成mRNA的前提，即核内不均一RNA（hnRNA），剪接加工后送往胞质作为合成protein的模版
  3. RNA聚合酶III：位于细胞核质内，离子强度范围宽，转录产物是tRNA、5SrRNA、snRNA(核内小 RNA)，snRNA参与RNA的剪接。

真核生物启动子对转录的影响

  1. 启动子：确保转录精确而有效地起始的DNA序列。真核生物有3类不同RNA聚合酶负责转录，这些 基因的启动子结构也有各自的特点。
  2. RNA聚合酶II催化转录的DNA序列5′上游区有一 段与原核生物Pribnow区相似、富含TA的保守序列。由于该序列前4个碱基为TATA，所以又称为 TATA区
  3. 绝大多数功能蛋白基因启动子（真核）在-25～-35区有TATA区，是核心启动子的组成部分
  4. 在核心启动子上游，存在转录调控区，有多个启动元件。将TATA区上游的保守序列称为上游启动子元件（UPE）包括：GC区和CAAT区
  5. TATA区：使转录精确地起始，除去后，产物突变较不明显，但起始点不固定
  6. UPE：控制转录起始频率，基本不参与起始位点的确定，CAAT区作用最大


原核和真核基因启动子和启动子上游区的区别

  1. 原核启动区较小，Pribnow区外只有TTGACA区作为RNA Pol结合位点，参与调控。
  2. 真核启动区，调控区较大，有CAAT区，GC区和增强子区

转录起始复合物的组装

  1. 真核生物RNA聚合酶不能直接识别基因的启动子区，需要至少7种被称为转录调控因子的辅助蛋白质（与σ因子在细菌转录中所执行的功能类似的）， 按特定顺序结合于启动子上，帮助RNA聚合酶特异性地结合到靶基因的启动子上并解开DNA双链。
  2. 转录因子和RNA聚合酶II必须以特定的顺序结合到 启动子序列上。形成转录前起始复合物的第一步，Pol的亚基与TATA区结合，其余酶，亚基结合
  3. RNA Pol II最大亚基C端结构域CTD重复序列有多个位点可以发生磷酸化，可以调节转录起始。CTD的磷酸化与非磷酸化的互变，对于转录延伸过程也具有重要的调节作用。

增强子及其功能

增强子：在SV40的转录单元上发现它的转录起始位点上游约200碱基对处有两段72碱基对长的重复序列，它们不是启动子的一部分，但能增强或促进转录的起始，除去这两段序列会大大降低这些基因的转录水平。可以强化转录起始。
增强子又称远上游序列
增强子特点：

  1. 远距离效应
  2. 无方向性
  3. 顺式调节
  4. 无物种，基因特异性
  5. 有组织特异性
  6. 有相位性
  7. 某些增强子可应答外部信号

RNA转录的抑制

转录抑制剂主要分三类：嘌呤和嘧啶类似物DNA，模板功能抑 制剂，RNA聚合酶的抑制物。

  1. 嘌呤和嘧啶类似物：可以直接抑制核苷酸合成有关的酶类，或者通过掺入核酸分子形成异常的核酸结构，从而影响核酸的进一步延伸。
  2. DNA模板功能抑制剂：与DNA结合，使DNA失去模版功能，从而抑制转录，如放线菌素D
  3. RNA聚合酶抑制剂：能抑制RNA Pol的活性的物质，如 α-鹅膏蕈碱

真核生物RNA转录后加工

真核生物RNA的内含子

真核基因大多是断裂的，一个基因可由多个内含子和外显子间隔排列而成
真核基因表达往往伴随着RNA的剪接过程从mRNA前体分子中切除被称为内含子的非编码区，并使基因中被称为外显子 的编码区拼接形成成熟mRNA。
不同生物细胞内含子边界存在相似的核苷酸序列


真核生物tRNA前体的转录后加工

tRNA内含子

真核生物tRNA基因有内含子，其前体必须经过剪接。其内含子具有下列特点:

  1. 长度和序列没有共同性
  2. 位于反密码子的下游;
  3. 内含子和外显子间的边界没有保守序列。
tRNA的加工
  1. 内含子的剪接，tRNA内切核酸酶切除前体中的内含子，RNA连接酶连接外显子
  2. 3‘端添加CCA，真核tRNA前体的3’端缺乏-CCAOH结构，在tRNA核苷酸转移酶作用下在3‘端添加CCA
  3. 核苷酸修饰，tRNA含有较多稀有核苷酸，修饰频繁

真核生物rRNA前体的转录后加工

  1. 大多数真核rRNA基因没有内含子（rRNA保守），有些有内含子但是不转录
  2. 新生rRNA前体与蛋白质结合，形成巨大的核糖核蛋白前体(pre-rRNA)颗粒，rRNA前体必须被剪接成成熟的rRNA分子，此过程在核仁中进行
  3. 核仁小RNA（snRNA）参与反应，进行识别确认切割位点
哺乳动物rRNA前体的剪切过程
  1. 切除5′端非编码的序列，生成 41S中间产物。
  2. 41S RNA被切割为两段，一段32S，含有28S rRNA和5.8S rRNA，另一段20S，含有18S rRNA。
  3. 32S RNA被剪切成 28S rRNA和 5.8S rRNA。
  4. 20S RNA被剪切生成 18S rRNA。

真核生物mRNA的剪接

  1. mRNA前体的剪接主要有3种:pre-mRNA剪接、I类和II类自剪接内含子。
  2. 由DNA转录生成的mRNA的前体，即核内不均一RNA(hnRNA)， 经过 5′加“帽”和 3′ 酶切加多（A）腺苷酸“尾”，再经过RNA的剪接，编码蛋白质的外显子部分就连接成为一个连续的可读框，通过核孔进入细胞质，作为蛋白质合成的模板。
  3. mRNA的组成：
    1. 5‘端上游非编码区（5’UTR），位于AUG之前
    2. 编码区，起始密码子AUG到终止密码子
    3. 3‘端下游非编码区（3’UTR），终止密码子后不翻译的区域
  4. 内含子两侧剪接位点不同：左边称供体位点，右边称受体位点
核mRNA的剪接
  1. 转录产生的核内mRNA前体分子与蛋白质结合，形成RNA和蛋白质组成的snRNP复合物
  2. 随着RNA链的延伸，每个内含子5’和3’两端的 复合物成对联结，产生60S的颗粒—剪接体 (spliceosome)，进行RNA前体分子的剪接。
RNA序列决定剪接发生位点
  1. GU-AG法则：mRNA前体中内含子的两端边界存在共同的序列， 这些序列结构可能是产生mRNA前体剪接的信号。 多数细胞核mRNA前体内含子的5′边界序列为 GU，3′边界序列为AG。
  2. GU-AG法则不适用于线粒体和叶绿体的内含子
  3. 除了边界序列外，内含子内部的部分序列也可能参与内含子剪接
RNA剪接中的两步转酯反应
  1. RNA剪接是由两步转酯反应完成的，这两步转酯反应使得pre-mRNA中原有的某些磷酸二酯键断开，并形成一些新的磷酸二酯键。
  2. 转酯反应中，没有增加新的化学键，只是断开 了两个磷酸二酯键，同时形成了两个新的磷酸 二酯键。
    剪接体与RNA剪接
  3. 转酯反应是由一个被称为剪接体的大型复合体介导的。
  4. 剪接体是mRNA前体在剪接过程中组装形成的多组分复合物，是一种具有催化剪接反应的核糖核蛋白复合体（snRNP），它包含约150种蛋白质和5种RNA，大小与核糖体差不多。剪接体中的5种 RNA(U1、 U2、U4、U5 和 U6)统称为核小 RNA(snRNA)
  5. snRNP在剪接中的功能：
    1. 识别5′ 剪接位点和分支点;
    2. 按需要把这两个位点集结到一起;
    3. 催化或协助催化RNA的剪接和连接反应。
I类和II类自剪接内含子

I、II类内 含子，因为带有这些内含子的RNA本身具有催化活性核酶，能通过自身折叠成一种特殊的构象来进行内含子的自我剪接，而不需要形成剪接体。
I类内含子：真核细胞核前体rRNA、细胞器RNA、少数细菌RNA
II类内含子：细胞器RNA、部分细菌RNA
